== Description ==

This plugin implements a payment gateway for WooCommerce to let buyers pay with pay with BUX tokens on eCash, via PayPal or Credit Card.

== Installation ==

1. Upload the `bux-payment-gateway-for-woocommerce` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. In the WooCommerce Settings page go to the Payment Gateways tab, then click BUX.digital.
1. Check "Enable BUX.digital" and enter your eToken address. If you do not have an eCash wallet, you can create one at wallet.badger.cash.
1. Click "Save changes" and the gateway will be active.
